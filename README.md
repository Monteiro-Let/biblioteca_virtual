## 📚 Introdução <img src="https://cdn-icons-png.freepik.com/256/3446/3446226.png" width="25">

A **Biblioteca Virtual** é um projeto criado originalmente com WordPress que simula um acervo digital com o objetivo de incentivar a leitura gratuita, prática e acessível.
Para fins educacionais e de portfólio, o projeto foi desenvolvido para o fórum temático de Hands on Work X, da Universidade do Vale do Itajaí.

🔗 **Acesse o projeto online:**  
👉 [Clique aqui para visualizar a Biblioteca Virtual](https://biblioteca-virtual.free.nf/)

---

## 🚀 Funcionalidades <img src="https://cdn-icons-png.freepik.com/256/1850/1850648.png" width="25">

- Catálogo de livros organizado por gênero
- Leitura online integrada ao navegador
- Layout adaptável para dispositivos móveis
- Navegação simples e intuitiva

## 🔍 Review do Projeto <img src="https://cdn-icons-png.freepik.com/256/16991/16991620.png" width="25">

O foco foi criar uma interface limpa, funcional e acessível para todos os públicos, com seções bem definidas e navegação fluida.  
A escolha de cores suaves, ícones ilustrativos e uma estrutura objetiva oferece uma experiência agradável ao usuário.
